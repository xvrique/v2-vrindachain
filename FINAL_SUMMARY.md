

## 🎯 Customization

### Change Colors
Edit `tailwind.config.js`:
```js
colors: {
  'saffron': '#your-color',
  'gold': '#your-color',
}
```

### Update Content
Edit component files in `src/components/`:
- Hero text, buttons, stats
- Technology descriptions
- Tokenomics data
- Roadmap phases
- Community links

### Add New Sections
1. Create component in `src/components/`
2. Import in `src/App.tsx`
3. Add to JSX
