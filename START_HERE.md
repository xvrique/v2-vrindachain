## 🆘 Troubleshooting

**Port 5173 already in use?**
```bash
npm run dev -- --port 3000
```

**Build fails?**
```bash
rm -rf node_modules dist
npm install
npm run build
```

**Styles not loading?**
- Clear browser cache
- Restart dev server
- Check `src/index.css` is imported
